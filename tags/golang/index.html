Golang 切片

March 15, 2020

最近在使用 golang 创建子集的时候，遇到了一些问题，下面是代码：

golang
func Subsets(nums []int) [][]int {
    var sets = make([][]int, 0)
    var t = make([]int, 0)
    sets = append(sets, t)
    for i := 0; i < len(nums); i++ {
        for _, v := range sets {
            t = append(v, nums[i])
            sets = append(sets, t)      }
    }
    return sets
}

当测试切片的长度不大于 4，比如 []int{1,2,3,4} 的时候，程序输出正常，结果也是正确的。
不过当切片长度大于 4 的时候，比如 []int{1,2,3,4,5}，程序输出的二维切片的长度虽然正确，但是期中有一些数据就会有问题。在 debug 的时候发现，当计算完 sets[21]，开始计算 sets[22] 的时候，sets[15] 会从 []int{1,2,3,4}，变成 []int{1,2,3,5}。

Goalnd 基础配置

October 30, 2019

首先先设置一下 goproxy 和 go module。在系统的环境变量当中添加以下两句（zsh 的配置环境在 ~/.zshrc，bash 的在 ~/.bashrc）:

shell
export GO111MODULE=on
export GOPROXY=https://goproxy.cn
export PATH=$PATH:$HOME:/go/bin

在 goland - settings - go - go modules，中，选中 Enable go Module，并在 Proxy 中填入代理站点，如 https://goproxy.io 或者 https://goproxy.cn。
