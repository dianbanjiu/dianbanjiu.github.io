Windows配置nginx

创建于 2024-06-13 | 更新于 2024-06-13

1、安装 nginx
我是通过 scoop 安装的 nginx

scoop update
scoop install nginx

2、测试 nginx

nginx -p "$env:NGINX_HOME"

命令执行成功之后，在浏览器输入 http://127.0.0.1:80 就可以看到 nginx 的输出信息了
